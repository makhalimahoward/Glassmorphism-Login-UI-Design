Glassmorphism Login UI

A sleek, modern, and highly aesthetic Login Interface designed with the Glassmorphism trend. This project focuses on clean code and the frosted glass effect using pure HTML and CSS.

Live Demo
You can see the design in action here: 
https://makhalimahoward.github.io/Glassmorphism-Login-UI-Design/

Features
- Frosted Glass Effect: Achieved using CSS backdrop-filter.
- Fully Responsive: Works perfectly on Mobile, Tablets, and Desktops.
- Modern Typography: Clean and readable font integration.
- Hover Effects: Interactive buttons and input fields.

Tech Stack
- HTML5: Semantic structure.
- CSS3: Custom properties, Flexbox, and Glassmorphism styling.

How to Use
To use this design in your own project:
1. Clone the repository or download the ZIP file.
2. Open index.html in your browser.
3. Feel free to customize the colors and blur intensity in style.css!
